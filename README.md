# Suitmedia | Ideas Page - Frontend Test

## Project Description

This is a frontend implementation for the Suitmedia Frontend Developer internship test. It is built using React.js and Vite. The project focuses on building user interface, fetching API data, and managing state.

## Technologies Used

- React.js
- Vite
- React Router DOM
- Lucide React

## Prerequisites

Before you begin, make sure you have the following installed:

- Node.js (version 16 or newer)
- npm or yarn

## How to Run locally

1. Clone this repository.

2. Navigate to the project directory:

   ```bash
   cd suitmedia-frontend-intern-test
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

   _(or `yarn install` if you use yarn)_

4. Run the development server:

   ```bash
   npm run dev
   ```

   _(or `yarn dev`)_

5. Open your web browser and go to the URL shown in your terminal (usually `http://localhost:5173`).

## Project Structure

- `public/`: Static files that are not processed by the bundler.
- `src/`
  - `assets/`: Images, icons, and fonts.
  - `components/`: Reusable React components (such as Banner, Header, ListPost, and Pagination).
  - `pages/`: Page components (About, Careers, Contact, Services, Work).
  - `utils/`: Helper functions (like format.js).
  - `App.jsx`: The root component.
  - `main.jsx`: The entry point of the application.
